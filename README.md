# AirSense 🌬️

## Overview
AirSense is a hyperlocal Air Quality Index (AQI) dashboard that 
translates real-time satellite and weather data into interactive 
heatmaps and actionable health recommendations — helping 
vulnerable individuals make smarter daily decisions.

## Problem
City-wide AQI data is too broad for daily decision-making, leaving 
runners, parents, and asthmatics blind to hidden pollution pockets 
in their neighborhood.

## Solution
AirSense bridges that gap by providing:
- 🗺️ Real-time hyperlocal AQI heatmaps
- ⏱️ 24–72 hour air quality forecasts  
- 🔔 Mock push notifications for critical pollution thresholds
- 👤 Persona-based health alerts (Runner, Asthmatic, Parent, Elderly)

## Tech Stack
- Frontend: React.js
- Maps: Leaflet.js / Mapbox
- APIs: OpenWeatherMap, Open-Meteo
- Database: Firebase
